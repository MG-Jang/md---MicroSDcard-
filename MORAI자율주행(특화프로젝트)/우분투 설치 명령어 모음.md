# 참고 사항

- "-" 가 복붙을 하는 경우 모양만 같고 다른 "-" 로 입력 되는 경우가 있음. 
- 따라서 실행이 정상적으로 안되는 경우 - 를 삭제후 키보드에서 타이밍으로 입력 해보자

```py
$ sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'
$ sudo apt install curl
$ curl -s https://raw.githubusercontent.com/ros/rosdistro/master/ros.asc | sudo apt-key add -
$ sudo apt update

# 가장 오래 걸리는 구간
$ sudo apt install ros-melodic-desktop-full   
$ sudo apt-get install python-rosdep
$ sudo rosdep init
$ rosdep update
# 아래 명령어는 bashrc 파일안에 "source /opt/ros/melodic/setup.bash" 문구를 추가한다는 뜻
# 만약 여기서 error 가 생긴다면  sudo vi bashrc 확인해 볼것
# 그리고 pdf 명세서에 나와있는 따옴표 표시가 잘 못되었으므로 만약 안된다면 ""를 타이핑으로 변경해 볼것(아래 명령어는 이미 변경된 상태)
$ echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc    
$ source ~/.bashrc
$ sudo apt install python-rosinstall python-rosinstall-generator python-wstool build-essential
$ sudo apt-get install ros-melodic-velodyne

# ROS가 정삭적으로 설치되었는지확인 단 아래 명령어가 실행되어도 rqt rviz는 실행되지 않을 수 도 있다.
$ roscore

#워크 스페이스, 패키지 생성 및 빌드
```
