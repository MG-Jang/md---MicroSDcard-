# 무드등 구현
작성자: 서울 21반 embedded system 장명근

> 개요
  - 기능구현
  - 가격
  - 디자인
  - 펌웨어 엄데이트 OTA
  - 전원공급 슬립모드

> 기능 구현

1. 가변저항(돌리는 스위치 형태)로 무드등의 밝기를 조절 한다.
2. LCD 창을 이용하여 현재 온도 및 습도를 나타낸다.
3. 초음파 센서를 이용하여 근처에 오면 스피커로 안녕하세요를 말해준다.
4. 스피커의 성능을 좋게 하여 하이엔드 스피커로도 사용이 가능하게 설계한다.

> 가격
1. ESP32 : 10000
2. 가변저항: 1000원
3. 0.95인치 LCD: 47960
4. 하이엔드 스피커: 50000
5. 스위치: 500
6. 초음파 센서: 1,100
