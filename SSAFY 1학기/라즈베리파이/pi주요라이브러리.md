> virtualenv(가상환경)
- 가상환경을 설치 하는 이유는 내가 2개 이상의 프로그램을 실행 시키는 경우 한쪽은 python 3.9를 지원하고 한쪽은 3.9를 지원하지 않는경우 각각을 실행하는 경우 계속해서 버전을 바꿔줘야 한다. 또한 각각의 라이브러리의 호환성을 위해 독립된 가상환경을 만든다.
- 명령어
```
$ sudo apt-get install python3-pip python3-virtualenv
```
